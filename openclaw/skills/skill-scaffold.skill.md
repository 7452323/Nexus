# Skill Scaffold — OpenClaw 技能脚手架生成器

根据用户需求自动生成规范的 SKILL.md 文件。

## 模板结构

```markdown
# [技能名称] — [一句话说明]
#
# 功能: [详细描述]

## 触发条件

- 自动触发: [什么情况下自动激活]
- 手动触发: 说"[关键词]"

## 执行步骤

1. [步骤说明]
2. [步骤说明]

## 检查清单

- [ ] [检查项]
- [ ] [检查项]

## 配置项

|名称|类型|默认值|说明|
|---|---|---|---|
|xxx|string|""|xxx|

## 适用场景

- [场景 1]
- [场景 2]
```

## 生成流程

```
用户说"帮我写个XX技能"
  -> 问功能: "这个技能做什么的？"
  -> 问触发: "手动还是自动？关键词？"
  -> 问输入输出: "需要什么输入？输出什么？"
  -> 生成 SKILL.md -> 保存到 skills/[技能名]/SKILL.md
  -> 告知用户文件位置
```
