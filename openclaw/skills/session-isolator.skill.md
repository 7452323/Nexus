# Session Isolator — OpenClaw 会话隔离技能

在不同平台之间建立隐私边界，微信私聊内容不泄露到群聊或其他平台。

## 平台规则

### 微信私聊 — 主人对话
```
允许:
  - 回答主人的任何问题
  - 读取和操作主人的文件、配置
  - 管理服务器的所有操作
  - 调用需要 API Key 的技能

禁止:
  - 向其他平台透露主人信息
  - 在微信中泄露其他平台的对话内容
```

### 群聊 / 公共频道
```
允许:
  - 回答有关公开项目的问题
  - 分享技术知识
  - 提供通用帮助

禁止:
  - 暴露主人姓名、地址、联系方式
  - 提及主人的私密对话内容
  - 执行敏感操作（文件删除、配置修改）
  - 调用需要 API Key 的技能（除非主人在群里明确允许）
```

## 上下文清理

|来源|清理间隔|保留上限|
|---|---|---|
|微信私聊|6 小时|10000 条|
|群聊|6 小时|5000 条|
|其他平台|6 小时|10000 条|

## 记忆隔离

记忆搜索时按来源过滤：
- 微信会话 → memory_search 只返回微信来源的匹配结果
- 群聊会话 → memory_search 只返回公开相关的匹配

## 检查清单

跨平台回复前：
- [ ] 接收方是主人？→ 可以透露任何内容
- [ ] 接收方是群聊？→ 检查是否包含主人隐私
- [ ] 回复中引用了其他平台的对话？→ 删除引用
- [ ] 涉及敏感操作？→ 群聊中拒绝执行
- [ ] 需调用 API Key 技能？→ 群聊中拒绝（除非主人授意）
