# Usage Analytics — OpenClaw 使用分析技能

统计和分析 OpenClaw 的使用数据，帮助了解运行状况。

## 统计项目

```
会话统计:
  每日会话数         -> 判断使用活跃度
  平均会话时长       -> 判断任务复杂度
  最活跃时间段       -> 优化定时任务时间
  会话来源分布       -> 微信/WebUI/API 占比

工具调用:
  每日工具调用次数   -> 计算工作负载
  最常用工具 Top 10  -> 了解高频功能
  工具调用成功率     -> 发现异常和错误
  平均响应时间       -> 性能基准

费用分析:
  每日 API 费用      -> 控制成本
  模型使用分布       -> 优化模型选择
  预估月费用         -> 预算规划
  7 天费用趋势       -> 发现异常波动
```

## 查询命令

|你说|结果|
|---|---|
|"使用统计"|显示今日统计数据|
|"本月费用"|显示本月 API 费用汇总|
|"最常用的工具"|显示工具调用频率排名|
|"什么时候最活跃"|显示各时段会话分布|
|"7 天趋势"|显示最近 7 天费用趋势|

## 数据存储

分析数据保存在 logs/analytics/ 目录下，按日期分文件。
