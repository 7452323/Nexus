# Rate Limiter — OpenClaw 请求频率控制技能

控制对外部 API 的请求频率，防止触发限流。

## 各 API 限流阈值

|API|每分钟上限|每日上限|触发后行为|
|---|---|---|---|
|DeepSeek|60 请求|10000 请求|等待 60 秒后重试|
|GitHub（匿名）|10 请求|—|等待 10 秒 + 指数退避|
|GitHub（带 Token）|30 请求|5000 请求|等待 5 秒 + 指数退避|
|通用 HTTP|120 请求|—|等待 2 秒|

## 指数退避

```
第 1 次触发 -> 等待 2 秒重试
第 2 次 -> 等待 4 秒
第 3 次 -> 等待 8 秒
第 4 次 -> 等待 16 秒
第 5 次 -> 等待 30 秒 + 通知用户
```

## 并发控制

|配置项|值|
|---|---|
|最大并发请求数|3|
|等待队列最大长度|20|
|超出队列行为|返回"队列已满，请稍后再试"|
