## 配置

|配置项|默认值|说明|
|---|---|---|
|推送时间|07:30|每天自动发送时间|
|包含费用|true|是否统计 API 费用|
|包含健康|true|是否包含服务器状态|
|包含活动|true|是否统计昨日活动|

## 触发方式

- 自动触发: 每天 07:30 自动生成并推送
- 手动触发: 说"日报"、"今天的报告"、"运行报告"
