# Config Presets — OpenClaw 配置预设切换技能

一键切换 OpenClaw 运行模式，应对不同场景。

## 预设模式

### 高性能模式
```
适用: 写代码、深度分析、大项目重构
模型: deepseek-v4-pro（推理拉满）
上下文: 100 万 token
并发: 4
超时: 120 秒
```

### 均衡模式（默认）
```
适用: 日常对话、普通任务
模型: deepseek-v4-flash
上下文: 50 万 token
并发: 2
超时: 60 秒
```

### 省电模式
```
适用: 简单聊天、快速查询
模型: deepseek-v4-flash（最快响应）
上下文: 20 万 token
并发: 1
超时: 30 秒
```

### 开发模式
```
适用: 调试脚本、测试新功能
模型: deepseek-v4-flash
上下文: 100 万 token
日志: DEBUG 级别
```

## 切换命令

|你说|效果|
|---|---|
|"高性能模式"|切换到高性能|
|"省电模式"|切换到省电|
|"均衡模式"/"默认"|回到默认|
|"开发模式"|开发模式（日志更多）|
