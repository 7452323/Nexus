# Memory Backup Auto — OpenClaw 记忆自动备份技能

定时打包记忆文件和关键配置，保留多版本快照，支持回滚。

## 备份策略

|快照类型|保留数量|触发时机|
|---|---|---|
|每日快照|最近 7 天|每天第一次会话时|
|每周快照|最近 4 周|每周一第一次会话时|

## 备份内容

|文件/目录|说明|
|---|---|
|`memory/*.md`|每日记忆文件|
|`MEMORY.md`|长期记忆|
|`TOOLS.md`|工具笔记|
|`RULES.md`|规则文件|
|`skills/*/learnings.json`|各技能的学习记录|

## 回滚流程

用户说"回滚记忆"时：

```
1. 列出可用快照（按时间倒序，显示日期和大小）
2. 用户选择回滚点（"回滚到 5月18日"）
3. 备份当前状态（防回滚错了还能再回）
4. 解压选定快照覆盖到 workspace/
5. 报告回滚结果
```

## 检查清单

备份前：
- [ ] memory/ 目录存在且可读
- [ ] 输出目录有足够空间（≥100MB）
- [ ] 文件未被其他进程锁定

备份后：
- [ ] 验证压缩包完整性（tar -tzf 测试）
- [ ] 文件数量匹配预期
- [ ] 清理 7 天前的旧快照
